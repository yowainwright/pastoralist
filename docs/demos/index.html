What Each Demo Shows

Basic: How pastoralist tracks overrides, creates documentation, and cleans up unused ones

Security: Scan for vulnerabilities, auto-fix issues, and manage security patches

Workspaces: Manage overrides across multiple packages in a monorepo

Migration: Transform existing manual overrides into documented, maintained ones

Run Locally

git clone https://github.com/yowainwright/pastoralist.git
cd pastoralist/demo/basic
npm install && npm start

Try These Commands

In any demo:

  • npm start - Run the demo
  • npm run check - See what pastoralist would do
  • npm run fix - Apply pastoralist fixes
