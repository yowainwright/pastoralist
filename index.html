Zero Maintenance Override Tracking

Pastoralist automatically creates an appendix that documents why each override exists. Track which packages depend on each override, detect security fixes, and clean up stale overrides when they're no longer needed.

  • Automatically tracks override dependencies
  • Documents security fixes with CVE references
  • Cleans up orphaned overrides automatically
  • Works with npm, yarn, pnpm, and bun
