## What _are_ \*overrides and resolutions?

> Overrides and resolutions solve the same problem.<br />
> They give developers a way to specify dependency versions downloaded to repository's node_modules folder.

Package manager CLIs solve this problem by adding an overrides or resolutions object to a repository's root package.json. This is awesome when fixing dependency issues with security and/or code issues.
