# Quiet Mode Example Pastoralist Sandbox

Demonstrates `--quiet` flag for CI pipelines using [Pastoralist](https://github.com/yowainwright/pastoralist).

## Setup

- minimist and lodash dependencies
- Security overrides configured

## Test

Run pastoralist and check exit code (0=clean, 1=vulnerabilities).

## Run

```sh
bun run start
```
