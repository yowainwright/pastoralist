# Security Scan Example Pastoralist Sandbox

Demonstrates `pastoralist --checkSecurity` flag for vulnerability detection using [Pastoralist](https://github.com/yowainwright/pastoralist).

## Setup

- Old versions of minimist, trim, lodash
- Security fix overrides

## Test

Run pastoralist with security scanning.
Check output for vulnerability reports.

## Run

```sh
bun run start
```
